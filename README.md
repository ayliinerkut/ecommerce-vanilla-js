# Modern E-Commerce (Vanilla JS)

Bu proje, **Vanilla JavaScript**, **HTML** ve **CSS** kullanılarak yapılmış bir modern e-ticaret frontend uygulamasıdır.  

FakeStore API kullanılarak ürünler gerçekçi bir şekilde çekilmektedir.  
Projede ürün listeleme, sepet sistemi ve responsive tasarım mevcuttur.

---

## 🛠️ Teknolojiler

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6+)  
- FakeStore API (https://fakestoreapi.com)  
- LocalStorage (sepet verisi için)  
- Responsive Tasarım  

---

## ⚡ Özellikler

- Ürünleri FakeStore API’den çekme  
- Ürün kartları ve fiyat gösterimi  
- Sepete ürün ekleme ve localStorage ile saklama  
- Responsive tasarım (mobil, tablet, desktop uyumlu)  
- Vanilla JS ile tamamen frontend odaklı
